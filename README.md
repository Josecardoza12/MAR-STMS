# MAR-STMS
Proyecto MAR-STMS
## Descripción del proyecto

El presente proyecto consiste en el desarrollo de un sistema de gestión basado en arquitectura de microservicios orientado a optimizar y automatizar procesos operacionales. El sistema integra distintos módulos independientes encargados de administrar etapas clave del flujo de trabajo, permitiendo una gestión centralizada y organizada de la información.

La solución considera procesos como recepción, órdenes de trabajo, diagnóstico, aprobación, reparación, inventario, pagos, bodega y seguimiento operacional, permitiendo controlar el ciclo completo de cada proceso dentro del sistema.

La arquitectura implementada busca mejorar la escalabilidad, facilitar el mantenimiento y reducir el acoplamiento entre componentes, mediante servicios independientes que se comunican a través de APIs REST.
